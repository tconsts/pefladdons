1. Создайте папку для скриптов в любом месте у себя на компьютере.
2. Сохраните туда скрипт http://pefladdons.googlecode.com/svn/trunk/dev/opera/sostav_na_match/peflsostavnamatchdev.js
3. В опере через меню откройте настройки Яваскрипт:

Preferences -> Advanced -> Content -> JavaScript Options...

4. В самом низу окошка с настройками есть возможность выбрать папку для пользовательских скриптов. Укажите там свою папку.
5. Откройте адрес http://www.pefl.ru/?team [для удобства рекомендуется добавить в закладки]
